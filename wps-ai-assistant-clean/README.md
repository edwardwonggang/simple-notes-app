# WPS AI Assistant Clean

全新实现的 WPS AI 对话插件。

- 项目目录：`/Users/edward/Documents/New project/wps-ai-assistant-clean`
- 本地开发地址：`http://127.0.0.1:3893/`
- 配置本地保存到 `config.local.json`
- 代理为可选项
- 侧栏不提供设置入口，配置统一手动编辑本地 `config.local.json`

直接编辑 `/Users/edward/Documents/New project/wps-ai-assistant-clean/config.local.json`，保存后重新打开 WPS 侧栏即可生效。
